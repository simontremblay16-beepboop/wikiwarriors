# Recent Fights Listing Frontend
The recent fights listings will be displayed as a sub-element of the index page. 
It will display a list of the 17 most recent non-daily fights, in order of when they occured.
Users will also be able to click on each entry to be presented with the fights results popup, corrisponding to the fight that was clicked.
